模型的体积、质量、质心位置和转动惯量计算

函数描述

计算模型的体积、质量、质心位置和转动惯量。

参数

  • π\piπ：圆周率。
  • LmmL_{\text{mm}}Lmm​：模型总长度（mm）。
  • Top0\text{Top0}Top0：顶点位置的无量纲量。
  • RnmmR_{\text{nmm}}Rnmm​：空化器半径（mm）。
  • NconN_{\text{con}}Ncon​：模型节数。
  • Part\text{Part}Part：每节的无量纲长度。
  • LfL_fLf​：前体长度（mm）。
  • ConeLen\text{ConeLen}ConeLen：每节的长度（mm）。
  • BaseDiam\text{BaseDiam}BaseDiam：每节的末端直径（mm）。
  • LhL_hLh​：腔体长度（mm）。
  • DLhD_{Lh}DLh​：腔体左直径（mm）。
  • DRhD_{Rh}DRh​：腔体右直径（mm）。
  • ρf\rho_fρf​：前体材料密度（kg/m³）。
  • ρa\rho_aρa​：后体材料密度（kg/m³）。
  • ρh\rho_hρh​：腔体材料密度（kg/m³）。

返回值

  • TMkgTM_{\text{kg}}TMkg​：模型总质量（kg）。
  • XcX_cXc​：质心位置（与模型长度相关）。
  • IcI_cIc​：质心处的转动惯量（kg*mm²）。

计算公式

1. 初始化变量

  • SumLen\text{SumLen}SumLen：每节的累积长度（mm）。
  • RRR：每节的半径（mm）。
  • QfwQ_{\text{fw}}Qfw​：前体各节的体积（mm³）。
  • QawQ_{\text{aw}}Qaw​：后体各节的体积（mm³）。
  • MfwM_{\text{fw}}Mfw​：前体各节的质量（kg）。
  • MawM_{\text{aw}}Maw​：后体各节的质量（kg）。
  • XciX_{\text{ci}}Xci​：各节的质心位置（mm）。

2. 计算顶点位置和空化器体积

Top=Top0×Lmm\text{Top} = \text{Top0} \times L_{\text{mm}}Top=Top0×Lmm​
Qn=π×Rnmm2×Top/3Q_n = \pi \times R_{\text{nmm}}^2 \times \text{Top} / 3Qn​=π×Rnmm2​×Top/3

3. 计算每节的长度和半径

SumLen[i]=Part[i]×Lmm\text{SumLen}[i] = \text{Part}[i] \times L_{\text{mm}}SumLen[i]=Part[i]×Lmm​
R[i]=BaseDiam[i]/2R[i] = \text{BaseDiam}[i] / 2R[i]=BaseDiam[i]/2

4. 找到前体末端的锥体编号

Lf>SumLen[i]L_f > \text{SumLen}[i]Lf​>SumLen[i]
Nf=iN_f = iNf​=i
NPf=2×i+4\text{NPf} = 2 \times i + 4NPf=2×i+4
NPa=2×Ncon−NPf+8\text{NPa} = 2 \times N_{\text{con}} - \text{NPf} + 8NPa=2×Ncon​−NPf+8

5. 计算腔体参数

RLh=DLh/2R_{Lh} = D_{Lh} / 2RLh​=DLh​/2
RRh=DRh/2R_{Rh} = D_{Rh} / 2RRh​=DRh​/2
Lhx=Lmm−LhL_{hx} = L_{\text{mm}} - L_hLhx​=Lmm​−Lh​
Rfh=RLh+(Lf−Lhx)×(RRh−RLh)/LhR_{fh} = R_{Lh} + (L_f - L_{hx}) \times (R_{Rh} - R_{Lh}) / L_hRfh​=RLh​+(Lf​−Lhx​)×(RRh​−RLh​)/Lh​

6. 计算每节的体积

6.1 前体部分

Lf≤ConeLen[0]L_f \leq \text{ConeLen}[0]Lf​≤ConeLen[0]
Rf=Rnmm+Lf×(R[0]−Rnmm)/ConeLen[0]R_f = R_{\text{nmm}} + L_f \times (R[0] - R_{\text{nmm}}) / \text{ConeLen}[0]Rf​=Rnmm​+Lf​×(R[0]−Rnmm​)/ConeLen[0]
Qfw[0]=π×Lf×(Rf2+Rnmm2+Rf×Rnmm)/3Q_{\text{fw}}[0] = \pi \times L_f \times (R_f^2 + R_{\text{nmm}}^2 + R_f \times R_{\text{nmm}}) / 3Qfw​[0]=π×Lf​×(Rf2​+Rnmm2​+Rf​×Rnmm​)/3
Qaw[0]=π×(ConeLen[0]−Lf)×(R[0]2+Rf2+R[0]×Rf)/3Q_{\text{aw}}[0] = \pi \times (\text{ConeLen}[0] - L_f) \times (R[0]^2 + R_f^2 + R[0] \times R_f) / 3Qaw​[0]=π×(ConeLen[0]−Lf​)×(R[0]2+Rf2​+R[0]×Rf​)/3

6.2 后体部分

Lf>SumLen[j]L_f > \text{SumLen}[j]Lf​>SumLen[j]
Rf=R[j−1]+(Lf−SumLen[j−1])×(R[j]−R[j−1])/ConeLen[j]R_f = R[j-1] + (L_f - \text{SumLen}[j-1]) \times (R[j] - R[j-1]) / \text{ConeLen}[j]Rf​=R[j−1]+(Lf​−SumLen[j−1])×(R[j]−R[j−1])/ConeLen[j]
Qfw[j]=π×(Lf−SumLen[j−1])×(Rf2+R[j−1]2+Rf×R[j−1])/3Q_{\text{fw}}[j] = \pi \times (L_f - \text{SumLen}[j-1]) \times (R_f^2 + R[j-1]^2 + R_f \times R[j-1]) / 3Qfw​[j]=π×(Lf​−SumLen[j−1])×(Rf2​+R[j−1]2+Rf​×R[j−1])/3
Qaw[j]=π×(SumLen[j]−Lf)×(R[j]2+Rf2+Rf×R[j])/3Q_{\text{aw}}[j] = \pi \times (\text{SumLen}[j] - L_f) \times (R[j]^2 + R_f^2 + R_f \times R[j]) / 3Qaw​[j]=π×(SumLen[j]−Lf​)×(R[j]2+Rf2​+Rf​×R[j])/3

7. 计算每节的质量

Mfw[i]=ρf×Qfw[i]/1000M_{\text{fw}}[i] = \rho_f \times Q_{\text{fw}}[i] / 1000Mfw​[i]=ρf​×Qfw​[i]/1000
Maw[i]=ρa×Qaw[i]/1000M_{\text{aw}}[i] = \rho_a \times Q_{\text{aw}}[i] / 1000Maw​[i]=ρa​×Qaw​[i]/1000

8. 计算总体积

Qf=∑QfwQ_f = \sum Q_{\text{fw}}Qf​=∑Qfw​
Qa=∑QawQ_a = \sum Q_{\text{aw}}Qa​=∑Qaw​
Q=Qf+Qa+QnQ = Q_f + Q_a + Q_nQ=Qf​+Qa​+Qn​

9. 计算腔体体积

Qh=π×Lh×(RLh2+RLh×RRh+RRh2)/3Q_h = \pi \times L_h \times (R_{Lh}^2 + R_{Lh} \times R_{Rh} + R_{Rh}^2) / 3Qh​=π×Lh​×(RLh2​+RLh​×RRh​+RRh2​)/3

10. 计算模型质量

Lf≠0L_f \neq 0Lf​=0
Mn=ρf×Qn/1000M_n = \rho_f \times Q_n / 1000Mn​=ρf​×Qn​/1000
Mf=ρf×Qf/1000M_f = \rho_f \times Q_f / 1000Mf​=ρf​×Qf​/1000
Ma=ρa×Qa/1000M_a = \rho_a \times Q_a / 1000Ma​=ρa​×Qa​/1000
Mh=ρh×Qh/1000M_h = \rho_h \times Q_h / 1000Mh​=ρh​×Qh​/1000

TMg=Mn+Mf+MaTM_g = M_n + M_f + M_aTMg​=Mn​+Mf​+Ma​

Lf≤LhxL_f \leq L_{hx}Lf​≤Lhx​
TMg=Mn+Mf+Ma−Qh×ρa/1000TM_g = M_n + M_f + M_a - Q_h \times \rho_a / 1000TMg​=Mn​+Mf​+Ma​−Qh​×ρa​/1000

11. 计算模型平均密度

ρ=1000×TMg/Q\rho = 1000 \times TM_g / Qρ=1000×TMg​/Q

12. 计算质心位置

Xc=(Xcf×(Mf+Mn)+Xca×Ma)/(Mf+Mn+Ma)X_c = (X_{\text{cf}} \times (M_f + M_n) + X_{\text{ca}} \times M_a) / (M_f + M_n + M_a)Xc​=(Xcf​×(Mf​+Mn​)+Xca​×Ma​)/(Mf​+Mn​+Ma​)

13. 考虑腔体

Lf≤LhxL_f \leq L_{hx}Lf​≤Lhx​
Xc=(Xc×(Mf+Ma+Mn)−Xch×(Qh×ρa/1000))/(Mn+Mf+Ma−(Qh×ρa/1000))X_c = (X_c \times (M_f + M_a + M_n) - X_{\text{ch}} \times (Q_h \times \rho_a / 1000)) / (M_n + M_f + M_a - (Q_h \times \rho_a / 1000))Xc​=(Xc​×(Mf​+Ma​+Mn​)−Xch​×(Qh​×ρa​/1000))/(Mn​+Mf​+Ma​−(Qh​×ρa​/1000))

14. 计算转动惯量

β=180∘\beta = 180^\circβ=180∘
Iof=0I_{\text{of}} = 0Iof​=0

Iof=Rnmm2×Top3×(0.2×Rnmm2/Top2+2/15)I_{\text{of}} = R_{\text{nmm}}^2 \times \text{Top}^3 \times (0.2 \times R_{\text{nmm}}^2 / \text{Top}^2 + 2 / 15)Iof​=Rnmm2​×Top3×(0.2×Rnmm2​/Top2+2/15)

Iof=∑(12×BaseDiam[i])2×((12×BaseDiam[i])2×ConeLen[i]+4×(Part[i]×Lmm)3−(Part[i−1]×Lmm)3)/3I_{\text{of}} = \sum \left( \frac{1}{2} \times \text{BaseDiam}[i] \right)^2 \times \left( \left( \frac{1}{2} \times \text{BaseDiam}[i] \right)^2 \times \text{ConeLen}[i] + 4 \times (\text{Part}[i] \times L_{\text{mm}})^3 - (\text{Part}[i-1] \times L_{\text{mm}})^3 \right) / 3Iof​=∑(21​×BaseDiam[i])2×((21​×BaseDiam[i])2×ConeLen[i]+4×(Part[i]×Lmm​)3−(Part[i−1]×Lmm​)3)/3

Iof=Iof×π×ρf/4×1012I_{\text{of}} = I_{\text{of}} \times \pi \times \rho_f / 4 \times 10^{12}Iof​=Iof​×π×ρf​/4×1012

Ioa=IofI_{\text{oa}} = I_{\text{of}}Ioa​=Iof​

Ioa=∑(12×BaseDiam[i])2×((12×BaseDiam[i])2×ConeLen[i]+4×(Part[i]×Lmm)3−(Part[i−1]×Lmm)3)/3I_{\text{oa}} = \sum \left( \frac{1}{2} \times \text{BaseDiam}[i] \right)^2 \times \left( \left( \frac{1}{2} \times \text{BaseDiam}[i] \right)^2 \times \text{ConeLen}[i] + 4 \times (\text{Part}[i] \times L_{\text{mm}})^3 - (\text{Part}[i-1] \times L_{\text{mm}})^3 \right) / 3Ioa​=∑(21​×BaseDiam[i])2×((21​×BaseDiam[i])2×ConeLen[i]+4×(Part[i]×Lmm​)3−(Part[i−1]×Lmm​)3)/3

Ioa=Ioa×π×ρa/4×1012I_{\text{oa}} = I_{\text{oa}} \times \pi \times \rho_a / 4 \times 10^{12}Ioa​=Ioa​×π×ρa​/4×1012

Io=Iof+IoaI_o = I_{\text{of}} + I_{\text{oa}}Io​=Iof​+Ioa​
